# Clustering Analysis on the Iris Dataset

A machine learning project that applies K-Means clustering to the Iris dataset to explore natural groupings within the data through unsupervised learning.

## Overview

This project demonstrates an end-to-end clustering workflow using the Iris dataset. It includes data exploration, visualization, feature analysis, determination of the optimal number of clusters, K-Means clustering, and cluster visualization.

The project highlights how unsupervised learning can identify inherent patterns in data without using target labels during training.

## Features

- Data exploration and preprocessing
- Exploratory Data Analysis (EDA)
- Pairplot visualization
- Elbow Method for selecting the optimal number of clusters
- K-Means clustering
- Cluster visualization
- Analysis of clustering results

## Dataset

- Iris Dataset
- Includes measurements of sepal length, sepal width, petal length, and petal width for three Iris species.

## Results

The project applies K-Means clustering to the Iris dataset and evaluates the resulting clusters using the Elbow Method. Visualizations help illustrate how the algorithm groups samples based on feature similarity and how those clusters relate to the underlying dataset structure.
