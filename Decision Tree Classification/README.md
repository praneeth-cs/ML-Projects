# Decision Tree Classification

A machine learning project that implements a Decision Tree classifier using scikit-learn to perform supervised classification on a structured dataset.

## Overview

This project demonstrates an end-to-end supervised machine learning workflow, including data exploration, preprocessing, model training, prediction, evaluation, and visualization using the Decision Tree algorithm.

The notebook highlights how decision trees learn hierarchical decision rules for classification while providing an interpretable machine learning model.

## Features

- Data exploration
- Data preprocessing
- Train-test split
- Decision Tree model training
- Model prediction
- Accuracy evaluation
- Classification Report
- Confusion Matrix
- Decision Tree visualization

## Dataset

- Structured classification dataset included with the project (`decisionTree_Data.csv`).

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Repository Contents

```text
├── decision_tree_classification.ipynb
```

## Results

The Decision Tree classifier is trained on the dataset and evaluated using standard classification metrics including accuracy, precision, recall, F1-score, and a confusion matrix. A visualization of the learned decision tree is also provided.
