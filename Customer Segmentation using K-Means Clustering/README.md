# Customer Segmentation using K-Means Clustering

A machine learning project that applies the K-Means clustering algorithm to segment customers based on purchasing behavior using the Mall Customers dataset.

## Overview

This project demonstrates an end-to-end unsupervised machine learning workflow, including data preprocessing, feature scaling, optimal cluster selection, clustering, visualization, and business interpretation.

The optimal number of clusters is determined using both the Elbow Method and Silhouette Analysis.

## Features

- Data preprocessing
- Feature scaling
- K-Means clustering
- Elbow Method
- Silhouette Analysis
- Cluster visualization
- Business insights from customer segments

## Dataset

- Mall Customers Dataset
- Includes customer demographics and annual spending information.

The optimal number of customer segments was determined using the Elbow Method and Silhouette Analysis. The resulting clusters provide meaningful insights into customer purchasing behavior and can be used to support targeted marketing strategies.
